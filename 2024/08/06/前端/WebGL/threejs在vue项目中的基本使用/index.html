官网：https://threejs.org/

Three.js 是一个跨浏览器的脚本，使用 JavaScript 函数库或 API 来在网页浏览器中创建和展示动画的三维计算机图形。

1 threejs 介绍

threejs 是运行在浏览器中的 3D 引擎，是 Javascript 编写的 WebGL 第三方库。提供了非常多的 3D 显示功能。开发者可以用它创建各种三维场景，包括了摄影机、光影、材质等各种对象。可以在它的主页上看到许多精彩的演示。

2.2.7 添加点光源 PointLight

创建点光源使用的是 PointLight，从 three 里引入

创建点光源需要有颜色、强度、照射距离、光照衰减值，以及位置

PointLight 有四个参数：

  • color -（可选参数）十六进制光照颜色。默认 0xffffff（白色）
  • intensity -（可选参数）光照强度。缺省值 1。
  • distance - 这个距离表示从光源到光照强度为 0 的位置。当设置为 0 时，光永远不会消失，默认 0。
  • decay - 沿着光照距离的衰退量。默认 1。

可以在官网查看其它光线使用

2.2.10 模型编辑逻辑梳理

模型的编辑，threejs 为我们提供了一个工具类叫做 变换控制器 TransformControls。它可以提供一种类似于在数字内容创建工具（例如 Blender）中对模型进行交互的方式，来在 3D 空间中变换物体。

点击要移动的模型，针对这个被点击的模型绑定一个 变换控制器，变换控制器 有三根轴，分别对应的就是 threejs 坐标系的 x 轴、y 轴、z 轴，拖动 变换控制器的轴，就可以实现对应模型的移动。

如何实现点击功能？

threejs 中提供 射线发射器 Raycaster 来实现，射线发射器 会根据鼠标在二维屏幕中点击的位置，结合当前相机的一些状态，比如位置、角度、方向等，从屏幕向鼠标点击的方向发出一条射线，把被射线穿过模型返回成一个列表回来，列表的顺序就是模型穿过的先后顺序。
